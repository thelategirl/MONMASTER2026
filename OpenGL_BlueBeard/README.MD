# BLUEBEARD OPENGL

## Description

Réalisation d'un teaser pour un jeu vidéo en cours de création.
BlueBeard (ou Barbe Bleue) est un jeu vidéo sur le lequel je travaille. L'histoire inspirée du fameux conte de Charles Perrault suit la progression d'une jeune femme mariée à un homme mystérieux et dangereux.
Ce jeu à la première personne est développé avec Unity.
